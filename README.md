# food_quality_monitoring_system
This project presents an IoT based food quality monitoring system designed to detect early signs of food spoilage. The system monitors environmental conditions such as gas concentration and temperature using sensors connected to a microcontroller. When abnormal conditions are detected, the system triggers an alert to notify users.

Features

* Real time monitoring of food storage conditions
* Detection of spoilage gases using a gas sensor
* Temperature monitoring using a temperature sensor
* Automatic alert when food quality decreases
* Helps reduce food waste and improve food safety

Components Used
ESP32 Microcontroller
Gas Sensor (MQ series)
DHT11 Temperature Sensor
Buzzer / Alert System
Jumper Wires
Power Supply

Technologies Used
Arduino IDE
Embedded C Programming
Internet of Things (IoT)

Working Principle
The gas sensor continuously monitors the presence of gases produced during food spoilage, while the temperature sensor measures the storage temperature. The ESP32 processes the sensor readings and compares them with predefined threshold values. If the gas level or temperature exceeds the safe range, the system activates an alert to warn the user about possible food spoilage.

Applications
Smart kitchen monitoring
Food storage safety systems
Cold storage monitoring
Food supply chain monitoring
